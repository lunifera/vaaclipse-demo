Welcome to Cassandra demo

Cassandra is the demonstration application for the Vaaclipse presenting some framework features.  Cassandra provides the IDE-like demo environment allowing you to browse its own code and learn some Vaaclipse features.

The Package Explorer view in the left side of screen contains the project with Cassandra source code. Using this view you can browse project and open project files.

The Help view in the right side of screen shows the Cassandra project help. Use this Help to examine the project structure and getting tips how to use Vaaclipse on the Cassandra example.
